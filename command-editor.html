派對遊戲 指令資料庫

編輯一般格 / 特別格指令，會存到瀏覽器（localStorage）

一般格指令

類型： 自行完成（抽到的人自己做）、 互動完成（會自動帶入 [A]/[B] 做互動）。
等級：A / B / C（遊戲格子前 40% 用 A，40% 用 B，最後 20% 用 C）。
可以使用 [A] / [B] 當人名變數，例如： [A] 跟 [B] 握手 10 秒。

特別格指令

會在踩到特別格時顯示，例如： 幸運格：你往前多走 1 格。
可以設定等級 A / B / C，會跟目前所在格子區段對應。

匯入 / 匯出 JSON

結構：{ normal: { text, kind, level }[], special: { text, level }[] }
資料會存到瀏覽器的 localStorage（partyGameCommandDB），遊戲頁面會自動讀取這個資料庫。
